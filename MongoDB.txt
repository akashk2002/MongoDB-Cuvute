# REST API's : CRUD Operations, HTTP Methods
    Crud Operations - HTTP Methods
    -  (C)Create - POST - .post()
    -  (R)Read - GET - .get()
    -  (U)Update - PATHCH - .patch()
    -  (D)Delete - DELETE - .delete()

# Mongoose Methods
    - GET : .find()
    - POST : .create()
    - Update : .findByIdAndUpdate()
    - Delete : .findByIdAndDelete()

-   Eg:
    -   Students
        - Read : Get the list of all students - /students
        - Create : Add a new student - /students + data
        - Update : Update the details of a student - /students/:id+data
        - Delete : Delete the student fron the school DB - /students/:id

Middleware 
    - body-parser : parse incoming request bodies in a middleware before your handlers, available under the req.body property
